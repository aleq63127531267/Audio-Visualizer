# Audio Visualizer Tutorial Content

## 1. Intro (Language Selection Embedded)
**Title:** Welcome!
**Text:** The box itself will explain how different parts of the window work. Follow the highlights to learn the interface.
*(Note: Language options are displayed here for quick setup)*

## 2. Global Buttons
**Title:** Global Controls
**Text:** Play, pause, and control master volume for all layers and backgrounds from here.

## 3. Visualizer Window
**Title:** The Stage
**Text:** This is your main workspace. Visualizers appear here. You can also drag and drop audio files directly onto this window.

## 4. Side Menu Button
**Title:** Visualizer Menu
**Text:** Click this button to open the Side Panel, where you can manage Layers and Recordings.

## 5. Side Menu Overview
**Title:** Side Panel
**Text:** This panel contains everything you need to manage your visualizers and saved recordings.

## 6. Visualizers Section
**Title:** Visualizers Section
**Text:** Manage your active visualizers here. Add new ones, select layers to edit settings, or use the "Select EQ/Struct" shortcuts.

## 7. Recordings Section
**Title:** Recordings Section
**Text:** Access your saved video recordings here. You can filter by Used/Unused, download, or delete them.

## 8. Main Controls
**Title:** Main Controls
**Text:** Use these buttons to customize colors, open settings, enter Record Mode, and control individual file playback.

## 9. Live Input
**Title:** Live Input
**Text:** Switch between Microphone and Desktop Audio input here. Toggle the mic on or off.

## 10. Top Menu
**Title:** Top Menu
**Text:** Change the language, access Experimental Features (like Transform Edit), or restart this tutorial.
